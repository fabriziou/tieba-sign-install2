详细安装教程请戳[创世神域](http://www.puteulanus.com/?p=442)。

    小清新透明风格-0-1-1

    自动领取T豆-0-1-8

    客户端回帖-0-2-3

    邮箱验证插件v1.1.2 

    高级管理面板-0-1-3
